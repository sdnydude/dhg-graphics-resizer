# Changelog

## V1.5 — 2026-02-12

### Added
- General-purpose graphics resizer (converted from headshot-only tool)
- 3 AI background removal workflows: BiRefNet-Portrait, BiRefNet-General, BRIA RMBG
- Tuned per-model alpha refinement (blur radius, threshold, alpha boost)
- Multi-workflow comparison — select multiple models, outputs organized into subfolders
- 40+ size presets: headshots, social media profiles/posts/banners, IAB digital ads, email, web
- Background replacement: solid colors, multi-stop gradients (down/right/diagonal/radial), transparency
- Brand presets: NACE Brand Gradient, NACE Full Gradient, ONA Teal, ONA Summit Gradient
- Custom gradient syntax with direction control
- Zoom-to-fill portrait scaling with top-aligned crop mode
- Virtual environment launchers for Mac (.command) and Windows (.bat)
- Auto-dependency installation on first run
- Mac tkinter detection with Copy-to-clipboard brew command
- JPEG/PNG/WebP export with quality control
- DHG-branded Word user guide (docs/)

## V1.0 — 2026-02-11

### Added
